  </Accordion>

    <Accordion title="404 - Not found" icon="circle-question">

    - O id inserido não existe ou o usuário não completou o registro de sua conta (não vinculou o Riot ID)
    **Queue endpoints**
    - A fila não existe
  </Accordion>
